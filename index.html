1. SELECTORES

1. SELECTORES

Ejemplo más recomendado (siempre):
cy.get('[data-testid="miElemento"]');

Usar el atributo data-testid es una excelente práctica, común en frameworks como Cypress y Playwright, ya que es estable y no se ve afectado por cambios en el estilo o la estructura del HTML.

Ejemplo recomendado (siempre):
cy.get('[data-cy="miElemento"]');

Al igual que data-testid, el uso de data-cy asegura que los selectores sean aislados de cambios de estilo o comportamiento.

Por el texto: contains
Ejemplo (a veces recomendado):
cy.contains('Texto del Elemento');

Este enfoque depende del contenido del texto, que puede cambiar. Úsalo cuando el texto sea fijo y no es probable que se modifique.

Ejemplo por nombre del atributo (uso moderado):
cy.get('[name="miNombre"]');

Es útil si el atributo name está bien definido en el HTML, pero puede cambiar dependiendo del diseño de la página o el contexto.

Ejemplo por ID (uso restringido):
cy.get('#miElemento');

Los IDs pueden cambiar o ser usados por JavaScript, lo que los hace menos fiables que un selector por data-testid o data-cy.

Ejemplo por clase (evitar):
cy.get('.claseElemento');

Este enfoque depende del estilo y las clases CSS, que son propensas a cambiar con el tiempo.

Ejemplo por nombre del elemento (evitar):
cy.get('input');

Los selectores basados en nombres de elementos son demasiado genéricos y no proporcionan suficiente contexto, lo que aumenta la probabilidad de seleccionar el elemento incorrecto.

Obtener título de la página: title
Ejemplo:
cy.title().should('eq', 'Título Esperado');
Obtener URL de la página: url
Ejemplo:
cy.url().should('include', 'ejemplo.com');
Obtener elementos dentro de otros elementos: Parent
Ejemplo:
cy.get('.hijo').parent();

3. ASSERTIONS: VALIDACIONES

Assertions
Ejemplo 1: Verificar que un elemento es visible:
cy.get('#elemento').should('be.visible');
Ejemplo 2: Verificar que el texto de un elemento es correcto:
cy.get('.mensaje').should('have.text', '¡Éxito!');
Ejemplo 3: Verificar que un elemento tiene un valor específico:
cy.get('input').should('have.value', 'Texto de prueba');
Ejemplo 4: Verificar que un elemento contiene un texto específico:
cy.get('.alert').should('contain', 'Advertencia');
Ejemplo 5: Verificar que un elemento no está visible:
cy.get('#oculto').should('not.be.visible');
Ejemplo 6: Verificar que un elemento tiene una clase específica:
cy.get('.activo').should('have.class', 'activo');
Ejemplo 7: Verificar que un elemento está habilitado:
cy.get('#boton').should('be.enabled');
Ejemplo 8: Verificar que un elemento está deshabilitado:
cy.get('#boton').should('be.disabled');
Ejemplo 9: Verificar que un elemento es un enlace:
cy.get('a').should('have.attr', 'href', 'https://ejemplo.com');
Ejemplo 10: Verificar que el tamaño de una lista es correcto:
cy.get('.lista').children().should('have.length', 3);
